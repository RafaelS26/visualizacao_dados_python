# Visualização de Dados com Python

Projeto desenvolvido para praticar visualização de dados em Python,
com foco na criação de gráficos e dashboards para análise exploratória.

O objetivo é transformar dados em informações visuais claras,
facilitando a interpretação e a comunicação de insights.

## Tecnologias
- Python
- Pandas
- Matplotlib / Seaborn / Plotly (conforme utilizado)

## Como usar
1. Clone o repositório
2. Instale as bibliotecas necessárias
3. Execute o script Python para visualizar os gráficos
